## 📱 Refer-to-SMS Community Bot

একটি শক্তিশালী টেলিগ্রাম বট যা ব্যবহারকারীদের তাদের নম্বর লুকিয়ে রেখে যেকোনো নম্বরে SMS পাঠাতে সাহায্য করে।

### 🎯 মূল ফিচার

✅ **নম্বর গোপনীয়তা** - ভার্চুয়াল নম্বর থেকে SMS পাঠান  
✅ **বাধ্যতামূলক সদস্যপদ** - চ্যানেল ও গ্রুপে যোগ দিন  
✅ **রেফারেল সিস্টেম** - বন্ধুদের রেফার করে পয়েন্ট অর্জন করুন  
✅ **পয়েন্ট ভিত্তিক** - পয়েন্ট খরচ করে SMS পাঠান  
✅ **এসএমএস হিস্ট্রি** - সব পাঠানো SMS রেকর্ড রাখুন

### 🛠️ প্রয়োজনীয় জিনিস

- Python 3.8+
- Telegram Bot Token (BotFather থেকে)
- Twilio Account (SMS পাঠানোর জন্য)
- টেলিগ্রাম চ্যানেল ও গ্রুপ

### 📥 ইনস্টলেশন

1. **রেপোজিটরি ক্লোন করুন:**
```bash
git clone https://github.com/mdarifhosen186-commits/sms-bot.git
cd sms-bot
```

2. **ডিপেন্ডেন্সি ইনস্টল করুন:**
```bash
pip install -r requirements.txt
```

3. **`.env` ফাইল তৈরি করুন:**
```bash
cp .env.example .env
```

4. **`.env` ফাইল সম্পাদনা করুন:**
```env
TELEGRAM_BOT_TOKEN=your_bot_token_here
CHANNEL_USERNAME=@your_channel
GROUP_USERNAME=@your_group
CHANNEL_ID=-1001234567890
GROUP_ID=-1001234567890
TWILIO_ACCOUNT_SID=your_account_sid
TWILIO_AUTH_TOKEN=your_auth_token
TWILIO_PHONE_NUMBER=+1234567890
```

5. **বট চালু করুন:**
```bash
python bot.py
```

### 📖 কমান্ড ব্যবহার

| কমান্ড | বর্ণনা |
|--------|--------|
| `/start` | বট শুরু করুন |
| `/help` | সাহায্য পান |
| `/my_points` | আপনার পয়েন্ট দেখুন |
| `/my_referral` | রেফারেল লিংক দেখুন |
| `/stats` | পরিসংখ্যান দেখুন |
| `/sms <নম্বর> <বার্তা>` | SMS পাঠান |

### 📨 SMS পাঠানোর উদাহরণ

```
/sms +8801712345678 হ্যালো, এটি একটি টেস্ট SMS
```

### 💡 কীভাবে কাজ করে

1. **ব্যবহারকারী যোগ দেয়:** `/start` কমান্ড দিয়ে বট শুরু করে
2. **সদস্যপদ যাচাই:** চ্যানেল ও গ্রুপে যোগ দেয়
3. **রেফারেল লিংক:** অনন্�� রেফারেল লিংক পায়
4. **রেফার করে পয়েন্ট অর্জন:** বন্ধুদের লিংক শেয়ার করে পয়েন্ট পায় (প্রতি রেফারেল = ৫ পয়েন্ট)
5. **SMS পাঠায়:** পয়েন্ট খরচ করে SMS পাঠায় (প্রতি SMS = ১ পয়েন্ট)

### 📊 পয়েন্ট সিস্টেম

- **পয়েন্ট অর্জন:** প্রতিটি সফল রেফারেল = ৫ পয়েন্ট
- **পয়েন্ট খরচ:** প্রতিটি SMS = ১ পয়েন্ট
- **ন্যূনতম প্রয়োজন:** SMS পাঠাতে কমপক্ষে ১ পয়েন্ট থাকতে হবে

### 🗄️ ডেটাবেস স্ট্রাকচার

**users টেবিল:**
- user_id: টেলিগ্রাম ইউজার আইডি
- username: ইউজারনেম
- points: মোট পয়েন্ট
- referral_code: অনন্য রেফারেল কোড
- is_member: সদস্যতা স্ট্যাটাস

**referrals টেবিল:**
- referrer_id: যিনি রেফার করেছেন
- referred_user_id: যাকে রেফার করা হয়েছে
- created_at: সময়

**sms_history টেবিল:**
- user_id: ইউজার আইডি
- recipient_number: ফোন নম্বর
- message: বার্তা
- points_used: ব্যবহৃত পয়েন্ট
- status: স্ট্যাটাস (sent/failed)

### 🔒 নিরাপত্তা

- ব্যবহারকারীর আসল নম্বর কখনও প্রকাশ হয় না
- সব লেনদেন ডাটাবেসে রেকর্ড থাকে
- Twilio ব্যবহার করে নিরাপদ SMS পাঠানো

### 🐛 সমস্যা সমাধান

**বট সাড়া না দিলে:**
1. বট টোকেন সঠিক আছে কিনা চেক করুন
2. ইন্টারনেট সংযোগ যাচাই করুন
3. লগ বার্তা দেখুন

**SMS পাঠানো না গেলে:**
1. Twilio ক্রেডেনশিয়াল সঠিক আছে কিনা যাচাই করুন
2. ফোন নম্বর +দিয়ে শুরু হয়েছে কিনা চেক করুন
3. পয়েন্ট যথেষ্ট আছে কিনা দেখুন
